Ways to Communicate between Components
There are several way to communicate between components:
  • Input: The parent component can pass data to a child component via the @Input() decorator.
  • Output: The child component can send data to the parent component using the @Output() decorator and an EventEmitter.

Sibling Component Comminication
There are below way to communicate between sibling components:
  • Service: Components can communicate indirectly via a shared service. The service can store the data and use RxJS observables to allow components to subscribe and update the data.
  • ViewChild: Allows a parent component to access the child component or DOM element in the template.
